# dry

DRY up your stylesheets. Dry is a simple reverse-compiler for converting repetitious CSS declarations to LESS.

## Getting Started
Install the module with: `npm install dry`

```javascript
var dry = require('dry');
dry.awesome(); // "awesome"
```
